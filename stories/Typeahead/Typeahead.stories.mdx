# Typeahead

`Typeahead` is a built using `Dropdown`, `Dropdown.Menu`, `Dropdown.Item` and  `FormControl` that doubles up as a toggle.
`TypeaheadProps` extends `FormControlProps`.

## API

```jsx
import { Typeahead } from '@govtechsg/sgds-govtech-react/Typeahead'; 
```

```jsx
import {
  Typeahead,
  TypeaheadProps,
} from '@govtechsg/sgds-govtech-react/Typeahead';

export const TypeaheadComponent = () => {
  const menuList = ['apple', 'orange', 'banana'];
  return <Typeahead menuList={menuList} />;
};
```

## onChangeInput

`onChangeInput` is fired whenever the value of input changes whether due to typing directly in the input or from selection from menu.
The first argument of `onChangeInput` returns the string value of the input.
The second argument returns either the ChangeEvent or MouseEvent depending on the type of user interaction with `<Typeahead/>`

export const TypeaheadOnChangeTemplate = () => {
  const [fruits, setFruits] = useState('');
  const handleChange = (val, e) => {
    setFruits(val);
  };
  const menuList = ['apple', 'orange', 'banana'];
  return (
    <>
      <Typeahead
        menuList={menuList}
        onChangeInput={handleChange}
        label="Select a fruit"
      />
      <div>Selected fruits : {fruits}</div>
    </>
  );
};
