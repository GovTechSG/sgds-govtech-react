# Input Group

Easily extend `<Form.Control>` by adding text, buttons, or button groups on either side of textual inputs and custom select. Remember to place `<label>`s outside the input group.

export const Template = (args) => {
  const handleSelect = (e) => {
    setTitle(e);

  const [title, setTitle] = useState('USD');

## 2. Sizing

By default, form input elements will be set to medium size. Add `size="sm"` for small or `size="lg"` for large to `<InputGroup>` element and all elements within will resize accordingly.

        <Form.Control
          aria-label="Small"
          aria-describedby="inputGroup-sizing-sm"
        />
      </InputGroup>
      <br />

        <Form.Control
          aria-label="Large"
          aria-describedby="inputGroup-sizing-sm"
        />
      </InputGroup>
    </>
  </Story>

## 3. Checkboxes and radios

Use the `InputGroup.Radio` or `InputGroup.Checkbox` to add options to an input group.

## 4. Multiple inputs

While multiple inputs are supported visually, validation styles are only available for input groups with a single input.
