# Input Group

Easily extend `<Form.Control>` by adding text, buttons, or button groups on either side of textual inputs and custom select. Remember to place `<label>` outside the input group.

export const Template = (args) => {
  return (
    <>
      <Form.Label>Phone Number</Form.Label>
      <InputGroup className="mb-3">
        <InputGroup.Text id="basic-addon1">+65</InputGroup.Text>
        <Form.Control
          placeholder="Phone number"
          aria-label="Phone"
          aria-describedby="basic-addon1"
        />
      </InputGroup>
    </>
  );
};

</Canvas>

## 3. Sizing

By default, form input elements will be set to medium size. Add `size="sm"` for small or `size="lg"` for large to `<InputGroup>` element and all elements within will resize accordingly.

## 9. DropdownButton with Toggle Addons

An extended use case of `<DropdownButton>` add-on that can be customized for toggling value using the useState hook.

export const DropdownButtonToggle = (args) => {
  const [allInputs, setAllInputs] = useState({
    selectCurrency: 'USD',
    selectEmail: '@gmail.com',
  });
  const currencySelectHandler = (e) => {
    setAllInputs((prevState) => {
      return {
        ...prevState,
        selectCurrency: e,
      };
    });
  };
  const emailSelectHandler = (e) => {
    setAllInputs((prevState) => {
      return { ...prevState, selectEmail: e };
    });
  };
  return (
    <>
      <InputGroup className="mb-3">
        <DropdownButton
          variant="outline-secondary"
          title={allInputs.selectCurrency}
          id="input-group-dropdown-toggle-1"
          onSelect={currencySelectHandler}
        >
          <Dropdown.Item eventKey="USD">USD</Dropdown.Item>
          <Dropdown.Item eventKey="MYR">MYR</Dropdown.Item>
          <Dropdown.Item eventKey="JPY">JPY</Dropdown.Item>
          <Dropdown.Divider />
          <Dropdown.Item eventKey="SGD">SGD</Dropdown.Item>
        </DropdownButton>
        <Form.Control aria-label="Text input with dropdown button toggle left addon" />
      </InputGroup>
      <InputGroup className="mb-3">
        <Form.Control aria-label="Text input with dropdown button toggle right addon" />
        <DropdownButton
          variant="outline-secondary"
          title={allInputs.selectEmail}
          id="input-group-dropdown-toggle-2"
          onSelect={emailSelectHandler}
        >
          <Dropdown.Item eventKey="@gmail.com">@gmail.com</Dropdown.Item>
          <Dropdown.Item eventKey="@yahoo.com">@yahoo.com</Dropdown.Item>
          <Dropdown.Item eventKey="@outlook.com">@outlook.com</Dropdown.Item>
          <Dropdown.Divider />
          <Dropdown.Item eventKey="@zoho.com">@zoho.com</Dropdown.Item>
        </DropdownButton>
      </InputGroup>
    </>
  );
};

## 11. Segmented Buttons with Input Toggle

An example of a use case for split button is where it groups related commands together into a dropdown,
but also offers one-click access to a default choice that doesn’t require opening the menu.

export const SplitButtonToggle = (args) => {
  const [renderText, setRenderText] = useState('');
  const renderTextClickHandler = (e) => {
    setRenderText(e.target.textContent);
  };
  return (
    <>
      <Form.Label htmlFor="splitButtonInput">
        How did you know about us?
      </Form.Label>
      <Form.Text>Select an option from below.</Form.Text>
      <InputGroup className="mb-3">
        <SplitButton
          variant="outline-secondary"
          title="Email"
          id="segmented-button-dropdown-3"
          onClick={renderTextClickHandler}
        >
          <Dropdown.Item onClick={renderTextClickHandler}>Email</Dropdown.Item>
          <Dropdown.Item onClick={renderTextClickHandler}>
            Facebook
          </Dropdown.Item>
          <Dropdown.Item onClick={renderTextClickHandler}>
            Word of Mouth
          </Dropdown.Item>
          <Dropdown.Divider />
          <Dropdown.Item onClick={renderTextClickHandler}>
            Linkedin
          </Dropdown.Item>
        </SplitButton>
        <Form.Control
          aria-label="Text input with split button dropdown"
          id="splitButtonInput"
          defaultValue={renderText}
          readOnly
        />
      </InputGroup>
    </>
  );
};

## 12. Native Select Toggle

Use the `<Form.Select>` as the native select tag within the input group.
