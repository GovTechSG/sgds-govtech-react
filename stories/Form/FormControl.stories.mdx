# FormControl

Use FormControl for textual form controls like input and textarea. The default FormControl is an input element.
FormControl extends the HTMLInputElement and HTMlTextAreaElement and their attributes are applicable to FormControl as props.

<Story name="Default">{Template.bind({})}</Story>

<br />

## API

```jsx
import { FormControl } from '@govtechsg/sgds-react/Form';
// or
import { Form } from '@govtechsg/sgds-react/Form';
// and use <Form.Control />
```

## FormControl input with icon

Icons embedded in FormControl input provide users the visual cue on the input content.

Wrap `<FormControl/>` with `<FormControlGroup/>`. Only applicable for default `FormControl` using HTMl `<input/>`

Does not apply to textarea.

export const FormControlIcon = () => {
  return (
    <Form>
      <Form.Group className="mb-3" controlId="formControlWithIcon">
        <Form.Label>Search</Form.Label>
        <FormControlGroup icon={<i className="bi bi-search"></i>}>
          <FormControl />
        </FormControlGroup>
      </Form.Group>
    </Form>
  );
};
