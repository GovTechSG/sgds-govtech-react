## Accessibility consideration

If you are planning to group several checkboxes or radio buttons within a form  question, you should use `<fieldset>` and `<legend>`. This will ensure that assistive technologies — such as screen readers — will announce the related question when reading out the options. See explanation and examples at [w3.org: Grouping Controls](https://www.w3.org/WAI/tutorials/forms/grouping/).

## Validations

Use props `feedback`, `feedbackType`, `isValid`, `isInvalid` to show valid or invalid feedback.
See [Validations](/docs/components-form-validations--native-html-5-form-validation) story on how to use it in a form submission context.
