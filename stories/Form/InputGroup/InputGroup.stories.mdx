# Input Group

Easily extend `<Form.Control>` by adding text, buttons, or button groups on either side of textual inputs and custom select. Remember to place `<label>` outside the input group.

As our components are built on top of `React-Bootstrap` library, do check out the detailed `Form` API references <a href="https://react-bootstrap.github.io/forms/overview/" target="_blank">here</a>.

export const Template = (args) => {
  return (
    <>
      <Form.Label>Phone Number</Form.Label>
      <InputGroup {...args} className="mb-3">
        <InputGroup.Text id="basic-addon1">+65</InputGroup.Text>
        <Form.Control
          placeholder="Phone number"
          aria-label="Phone"
          aria-describedby="basic-addon1"
        />
      </InputGroup>
    </>
  );
};

## 2. Variant Icon Input

Users may choose to attach an icon within the`<Form.Control>` input. By default, `variant` string attribute is set to empty. You may add it to the `<InputGroup>` by setting it to `'has-icon`. To allow input add-on before the `<i>` tag, simply wrap a `<div>` onto the `<i>` & `<Form.Control>` as shown in the examples below.

## 3. Sizing

By default, form input elements will be set to medium size. Add `size="sm"` for small or `size="lg"` for large to `<InputGroup>` element and all elements within will resize accordingly.

## 10. Segmented Buttons with Input Toggle

An example of a use case for split button is where it groups related commands together into a dropdown,
but also offers one-click access to a default choice that doesn’t require opening the menu.

export const SplitButtonToggle = (args) => {
  const [renderText, setRenderText] = useState('');
  const renderTextClickHandler = (e) => {
    setRenderText(e.target.textContent);
  };
  return (
    <>
      <Form.Label htmlFor="splitButtonInput">
        How did you know about us?
      </Form.Label>
      <Form.Text>Select an option from below.</Form.Text>
      <InputGroup className="mb-3">
        <SplitButton
          variant="outline-secondary"
          title="Email"
          id="segmented-button-dropdown-3"
          onClick={renderTextClickHandler}
        >
          <Dropdown.Item onClick={renderTextClickHandler}>Email</Dropdown.Item>
          <Dropdown.Item onClick={renderTextClickHandler}>
            Facebook
          </Dropdown.Item>
          <Dropdown.Item onClick={renderTextClickHandler}>
            Word of Mouth
          </Dropdown.Item>
          <Dropdown.Divider />
          <Dropdown.Item onClick={renderTextClickHandler}>
            Linkedin
          </Dropdown.Item>
        </SplitButton>
        <Form.Control
          aria-label="Text input with split button dropdown"
          id="splitButtonInput"
          defaultValue={renderText}
          readOnly
        />
      </InputGroup>
    </>
  );
};
