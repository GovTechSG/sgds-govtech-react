# Action Card

Action Cards are Cards that are able to perform actions like selection or quantity toggling. Usually with built-in input elements.

## `<SelectableCard />`

`<SelectableCard />` are Cards with input checkbox , radio or switch functionalities.
`<SelectableCard />` is a composite component built with `<Card />`, `<Card.Body/>`, `<Card.Subtitle/>` and `<FormCheck />` with partially extended props from both Components.
Similar to `<Card />`, it is a wrapper that accepts subcomponents of `Card` , like `Card.Text`  and `Card.Subtitle` as children.

The `ref` of `<FormCheck />` is extended to the `<Card/>` making the entire `<SelectableCard/>` essentially an input

### Default

The default SelectableCard is a checkbox.

import {

  const [qty, setQty] = useState(1);

            name="test"
            categoryTitle={
              <>
                <i className="bi bi-box"></i>Laptop

            }
            type="radio"
            value="apple"
            checked={brand === 'apple'}
            onChange={handleChange}
            aria-label="selectablecard"
          >
            {AppleBody}
          </SelectableCard>
          <SelectableCard
            name="test"
            categoryTitle={
              <>
                <i className="bi bi-box"></i>Laptop
