# Quantity Toggle

`<QuantityToggle />` allow increments and decrement of value reflected on an `input`.

It is a custom component built from `InputGroup`, `Button`, `FormControl`

The default `QuantityToggle` requires `count` and `setCount` to be passed in as props.
The default step value is 1 which controls the increment / decrement value on every button click.
The `input` is of number type.

The minimum value for count is 0, if count < 0 is passed in, `QuantityToggle` will setCount back to 0
