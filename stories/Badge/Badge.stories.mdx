# Badges

## Component's Badge

Badge can also be used on other components as a dot indicator or text indicator for notification or alerts.

<Canvas>
  <Story
    name="Components Badge - dotIndicator"

  >
    {ComponentBadgeTemplate.bind({})}
  </Story>
  <Story
    name="Components Badge - textIndicator"
    args={{ textIndicator: true, bg: 'danger', children: '99+' }}
  >
    {ComponentBadgeTemplate.bind({})}
  </Story>

Add css selector `position-relative` to the Component that wraps the badge. See example below

```jsx
//dotIndicator
<Button className="position-relative">
   Inbox
   <Badge dotIndicator bg="danger">
   </Badge>
 </Button>
 //textIndicator
<Button className="position-relative">
   Inbox
   <Badge textIndicator bg="danger">
   </Badge>
 </Button>
```
